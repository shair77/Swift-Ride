# SwiftRide Ride Booking Platform

## Overview
SwiftRide is a ride booking platform developed using Java, Spring Boot, Hibernate, and MySQL. The system supports role-based workflows for Admin, Driver, and Passenger operations.

## Features
- User authentication and authorization
- Ride booking and cancellation
- Driver and passenger dashboards
- Payment and invoice management
- Real-time booking workflows
- Database-driven backend architecture

## Tech Stack
- Java
- Spring Boot
- Hibernate
- MySQL
- HTML/CSS/JavaScript

## Architecture
Frontend → Spring Boot Backend → Hibernate ORM → MySQL Database
